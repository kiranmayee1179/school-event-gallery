## 📌 About the Project

This is a responsive Event Photo Gallery Website designed to showcase school events like Annual Day, Festivals, Sports Day, and Classroom Activities.

---

## 🚀 Features

* Event-based categories
* Image gallery grid
* Lightbox image preview
* Responsive design
* Clean UI

---

## 🛠️ Tech Stack

* Base44 (No-code platform)
